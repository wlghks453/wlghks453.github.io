## ✨ 설치 방법

rubyinstaller-devkit-3.3.4-1-x64 를 설치

라이센스 체크 후 > 설치 > 설치 > 설치!! 건드린 것이 없음. 

console창이 뜨면 그냥 엔터를 쳐서 마무리를 합니다.

jekyll 와 bundler 설치
```
gem install jekyll bundler 
```

설치 후

```
bundle install
```
