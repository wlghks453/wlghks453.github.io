## 자식 메뉴 설정

```yaml
parent: Code
grand_parent: UI Components
```

parent: Code 만 하면 1뎁스 생성

grand_parent: UI Components  2뎁스 까지 가능합니다.

{: .warning }
> 2뎁스까지 생성할 경우 1뎁스 메뉴에서도 has_children 설정해야한다.
